Page Navigator Settings

Built-in Patterns

The extension automatically detects these common pagination patterns:

  • /page/2 or /pages/2
  • ?page=2 or ?pages=2
  • ?p=2

Custom Patterns

Add your own patterns in JSON format. Each pattern should have a pattern (regex) and replacement template:

Pattern Examples

Reddit-style:
{"pattern": "/offset/(\\d+)", "replacement": "/offset/$1"}
Search results:
{"pattern": "[?&]start=(\\d+)", "replacement": "?start=$1"}
Forum posts:
{"pattern": "/thread/\\d+/(\\d+)", "replacement": "/thread/123/$1"}

How to Use

Once configured, simply use the left and right arrow keys to navigate between pages on any website that matches your patterns.

The extension will automatically detect page numbers in URLs and increment/decrement them accordingly.
